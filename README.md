# Управление жизненным циклом ПО для системы сопровождения интеграции ИИ в общеобразовательную школу

## Описание проекта
Данный репозиторий содержит документацию и артефакты, связанные с описанием жизненного цикла программного обеспечения (ЖЦ ПО) для разрабатываемой системы. Система предназначена для организационно-педагогического сопровождения интеграции интеллектуальных обучающих платформ и генеративных ИИ-помощников в общеобразовательную школу.

## Основные артефакты
* **Документ с описанием ЖЦ ПО:** `Lifecyle-Management-AI-School.docx`
* **Диаграммы архитектуры:** См. папку `Diagrams/`
* **Матрица ответственности (RACI):** `Diagrams/RACI_Matrix.png`

## Связь с магистерской работой
Данный проект является частью магистерской работы по теме "Модель организационно-педагогического сопровождения интеграции интеллектуальных обучающих платформ и генеративных ИИ-помощников в общеобразовательную школу". Описание ЖЦ ПО демонстрирует практический подход к управлению разработкой и внедрением подобной системы.

## Как просмотреть
Для просмотра документации рекомендуется:
1. Открыть файл `Lifecyle-Management-AI-School.docx`.
2. Просмотреть диаграммы в папке `Diagrams/` (в форматах PNG и Drawio).
